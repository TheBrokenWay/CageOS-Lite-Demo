# Social assets

## Social preview image

Suggested primary preview image for the repository:

- `docs/assets/cageos-lite-demo-social-preview.png`

Purpose:

- GitHub social preview
- LinkedIn post image
- founder launch post attachment
- startup-program visual support

## Messaging alignment

This asset is meant to support one message:

"Governance outside the model."

It deliberately shows:

- a visible agent surface
- an independent governance layer
- bounded outcomes: allow, redirect, escalate, block

It deliberately avoids:

- internal enforcement mechanics
- sensitive architecture details
- exaggerated sci-fi branding
- security clichés that make the work look generic
